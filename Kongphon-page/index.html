เช็คชื่อเข้าเรียน

เกี่ยวกับผม

เป็นนักพัฒนาเว็บที่มีความคิดสร้างสรรค์และทักษะในการออกแบบหน้าเว็บไซต์ ใช้ HTML, CSS, และ JavaScript เพื่อทำให้ผลงานดูเป็นมืออาชีพ และใช้งานได้ง่ายบนอุปกรณ์ทุกประเภท

สนใจทำงานด้านเว็บไซต์ส่วนตัว ร้านค้าออนไลน์ หรือโปรเจคที่ช่วยให้ธุรกิจเติบโตได้เร็วขึ้น

ทักษะหลัก

HTML - สร้างโครงสร้างหน้าเว็บที่ชัดเจน
CSS - จัดแต่งสไตล์ให้หน้าเว็บสวยงามและตอบสนอง
JavaScript - เพิ่มการโต้ตอบ และฟังก์ชันการใช้งาน
Responsive Design - รองรับมือถือและแท็บเล็ต

ผลงานเด่น

เว็บไซต์โปรไฟล์ส่วนตัว

ออกแบบหน้า portfolio ที่แสดงข้อมูลประวัติ ทักษะ และผลงานพร้อม UX ที่อ่านง่าย

ระบบร้านค้าออนไลน์

พัฒนาเว็บขายสินค้าออนไลน์ พร้อมระบบตะกร้าสินค้าและฟอร์มติดต่อ

เว็บแอปจัดการงาน

สร้างแอปจัดการงานแบบง่าย ใช้ JavaScript และการจัดเก็บข้อมูลเบื้องต้น

ระบบเช็คชื่อด้วยโลเคชั่น

ระบบนี้ใช้ API ฟรีของเบราว์เซอร์เพื่อขออนุญาตเข้าถึงตำแหน่งปัจจุบัน และตรวจสอบว่าคุณอยู่ในบริเวณ มรภ.เทพสตรี ภายในรัศมี 1 กม.

จุดสีแดงคือจุดเช็คอิน มุมมองนี้แสดงตำแหน่งพื้นที่เช็คอินและระยะที่อนุญาต
